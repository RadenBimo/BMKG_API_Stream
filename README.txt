Program yang dibuat untuk menarik di 2 tanggal 30 dan 31 maret 2023.
1. jika folder kota kosong maka semua tanggal akan di pull.
2. jika folder kota berisikin minimal 1 file maka akan mengupdate dari tanggal terakhir didalam folder hingga paling update.
3. update akan menimpa di tanggal terakhir file dan menambahkan(update) file hingga tanggal paling baru , dengan pertimbangan jika API eror atau diambil pada waktu siang hari maka lebih baik tanggal terakhir pada folder ditimpah/replace karena untuk mendapatkan data pada jam terakhir
